Inherits from NSObject
Declared in GSFancyText.h
GSFancyText.m

Overview

A structure that stores tree based markup parsing result, and arrays of lines. @discussion The initialization can be from an unparsed markup string, or a parsed result tree. It also stores a dictionary of styles, which can be used in markup parsing, as well as style switching afterwards @note This is a model class. It only processes strings and generate arrays. To create a view, see GSFancyTextView
